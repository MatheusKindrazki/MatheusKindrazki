# Impeccable — Design Context

> One-time design brief for this project. All design work (`/impeccable craft`, `/impeccable teach`, and every sibling skill: animate, polish, layout, colorize, overdrive, etc.) must load and respect this file.

## Design Context

### Users

Primary audience is **international-first** (English by default, Portuguese as cultural tell):

1. **Founders & investors** evaluating Matheus as a co-founder / partner for ventures at MokLabs or Lugui.ai
2. **Engineering leaders at foreign companies** evaluating him for Principal / Staff / Founding-Engineer roles
3. **Engineers and technical peers** who land here through X, GitHub, or talks — potential collaborators, hires for future ventures, or authority signal

Context of use: mostly **desktop, focused evaluation sessions** (recruiter deep-dive, partner due diligence). Mobile is a secondary scan surface. Sessions are short — the interface has seconds, not minutes, to land the impression.

**Job to be done**: establish in the first 3 seconds that Matheus ("Kindra") is a **distinctive, high-agency builder** worth opening a conversation with. The portfolio IS the pitch — it must demonstrate taste and technical depth through the interface itself, not only through copy. Proof-of-craft happens on the canvas, not in a paragraph.

**Primary CTAs, in priority order**:
1. Open Jarvis chat (the live, personal AI built into the site — the strongest differentiator and the "don't believe me? ask jarvis" moment)
2. Email (`matheus@kindrazki.dev`)
3. Social follow (X / GitHub / LinkedIn)
4. Read `/now` for current focus

### Brand Personality

Three words: **confident · handcrafted · mechanism-minded**

- **Confident** — declarative statements, strong hierarchy, zero apologetic hedging. Typography carries weight. No "I'm just a..." copy.
- **Handcrafted** — every detail is considered: ornamental italic ampersand, off-axis subtitle rotation, square accent markers, scramble-reveal headlines, particle photo, grain drift, shimmer pass. Nothing is a framework default.
- **Mechanism-minded** — the interface exposes its machinery: index numbers (`01`), HUD chrome, status strip tickers, tabular numerals, uppercase tracked meta labels. Matheus's own philosophy is "mechanisms over magic" — make governance the aesthetic.

Voice: direct, English-first, occasional Portuguese routes (`/sobre`, `/projetos`, `/contato`) as cultural signature. Technical vocabulary without jargon-peacocking. Lowercase micro-labels, tight letter-spacing on meta.

Emotional goal: the visitor feels **"this person takes craft seriously and has taste"** within seconds. Intimidated-respect, not warm-fuzzy.

### Aesthetic Direction

**Style**: Editorial brutalism with a galactic edge. Raw typographic structure meets dimensional depth. Dark void as canvas, accents as signals/stars. Precision grid broken intentionally for emphasis.

**Theme**: **Dark-only — by design**.
- The black background IS the galactic void; the accents are stars and signals.
- Users consume in evening / late-night founder-and-engineer contexts.
- A light-mode toggle would dilute the positioning. Do not add one.

**References (feel captured)**:
- `rauno.me` — precision, motion craft, "every detail is considered"
- `linear.app` — mechanism-minded interface, tracked meta, engineered density
- `brittanychiang.com` — personal portfolio that doesn't feel like a portfolio template
- `pitch.com` — bold editorial typography
- `vercel.com` — confident minimalism with dimensional accents

**Anti-references (NEVER look like)**:
- Generic dev-portfolio boilerplate (icon grid of skills, gradient avatar circle, stacked "feature cards", "contact me ↓" below the fold)
- Corporate SaaS homepage (hero + 3 feature cards + testimonial carousel + CTA banner)
- AI-era slop: purple→blue gradients, glassmorphism on every surface, cyan-on-dark "sophisticated" palette, neon glow borders as decoration
- Flat Material/Bootstrap with a dark-mode toggle
- Anything that could plausibly be a Figma Community template

**Signature elements — SACRED, do not remove without explicit reason**:
- Particle photo canvas with explode-on-navigation transition (`ParticlePhoto`)
- Brutalist typographic compositions — rotated off-axis subtitles (`rotate(-2deg)`), ornamental italic serif ampersand in kindra-yellow, square geometric accent markers
- Scramble-in text reveals for headline words (`useScramble`)
- Slow grain drift overlay — never static, ~18s cycle
- Shimmer-pass on key accent characters (`shimmer-yellow`)
- HUD chrome: `01` index numbers, uppercase tracked labels, status strip ticker (`◈ MK · ...`)
- Tabular numerals and monospaced precision touches
- Exponential easing `cubic-bezier(0.19, 1, 0.22, 1)` everywhere (`--ease-smooth`) — never bouncy, never elastic
- Custom cursor on pointer-capable devices (`CustomCursor` + `.custom-cursor-active`)
- Left-aligned compositions, asymmetry, and deliberate grid breaks for emphasis
- Jarvis chat ("don't believe me? — ask jarvis") as a visible differentiator, not hidden in a corner

**Color system — PRESERVE**:
- Void: `#000000` (true black, intentional) + `#1c2023` for panels
- Text: body `#dddddd`, emphasis `#ffffff`, meta scale `#9a9a9a / #666 / #3f3f3f / #3a3a3a / #3a3a3a`
- Accents (signals):
  - `--color-kindra-yellow` `#e0a458` — primary accent / star
  - `--color-kindra-green` `#419d78`
  - `--color-kindra-red` `#d9594c`
  - `--color-kindra-blue` `#53a2be`
- Each nav link carries its own accent color as identity (`/projetos` = green, `/skills` = blue, `/sobre` = yellow, `/contato` = red). Respect this mapping when extending.

**Typography — PRESERVE**:
- Display: **Roboto Slab 700** (`--font-heading`) — serif with weight and presence
- Body: **Montserrat 400 / 700** (`--font-body`)
- Patterns:
  - Fluid `clamp()` on display type only (marketing surfaces)
  - Tracked uppercase for micro-labels: 9–12px, letter-spacing 0.18–0.3em
  - Italic serif reserved for ornamental accents (ampersand, "at")
  - Tabular numerals for index/HUD numbers
  - Body line length capped at readable ranges (max-w-[520–560px] on prose)

### Design Principles

Five rules that gate every design decision in this project:

1. **The interface IS the proof.** Every surface demonstrates craft; never let copy carry the message alone.
2. **Mechanisms on display.** Expose structure and process — index numbers, HUD chrome, status tickers, tabular digits. The machinery is part of the aesthetic.
3. **Dark as canvas, accent as signal.** The void is intentional; color is rationed and directional, never decorative.
4. **Brutalist structure, exponential motion.** Raw skeleton and asymmetry, but motion is always smooth `ease-out` (cubic-bezier 0.19, 1, 0.22, 1). Never bouncy. Never abrupt. Never linear.
5. **Reduce to essence, then one deliberate deviation.** Every composition has a clear skeleton plus one intentional break — rotated subtitle, ornamental ampersand, off-axis marker. Never symmetrical-default, never decoration-for-decoration.

**And one meta-rule**: reject any choice that would let a visitor think *"this could be any portfolio."* If it fits anywhere else, it does not belong here.
